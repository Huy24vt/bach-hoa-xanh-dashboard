Policy rules

  • Eligibility: attendance ≥ 95%, audit ≥ 85, stock availability ≥ 96%, on-time opening ≥ 95%.
  • Revenue factor: 1.2 if attainment ≥ 105%, 1.0 if 100%–104.99%, 0.6 if 95%–99.99%, else 0.
  • Shrinkage factor: 1.1 if shrinkage ≤ 0.5%, 1.0 if 0.5%–0.7%, 0.7 if 0.7%–0.9%, else 0.
  • Productivity factor: 1.1 if productivity ≥ 105% of target, 1.0 if 95%–104.99%, else 0.5.
  • Payout: base bonus budget × bonus index, only when all gates are passed.
